JavaScript and its relationship to HTML and CSS.
Consider a website consisting of a few essential components. These include HTML, CSS and Javascript. HTML provides the structure of the webpage, CSS provides the style and Javascript allows the user to interact with the webpage. An example would be that HTML creates a navigation menu, CSS adds style for instance changing colour when howevered and Javascript allow interaction for example running a script when a button is pressed.

Explain control flow and loops
Control flow describes how your code operates based on the inputs provided. They consist of if statments and loops, if statements conduct a tasks based on if a certain critera is met. Loops iterate through a task a certain amount of times. An example would be watering the plants on specific days until the plants have a certain amount of water. The specific days would be with if statements and the watering until a certain amount of water is provided would be with loops.
Describe what the DOM is and an example of how you might interact with it.
The DOM stands for the Document Object Model and allows users to interact an manipulate HTML documents. You can interact with it by adjusting HTML elements.
Explain the difference between accessing data from arrays and objects.
Objects have characteristic that can be accessed whilst arrays store a list of information. An example would be an object storing name and career data of individuals whilst an array would consist of only the salaries of individuals.
Explain what functions are and why they are helpful.
Functions allow a task to be repeated many times after delclaring the function. This allows time to be saved as code does not need to be rewritten.
