What is the difference between Margin, Border, and Padding?

Consider every object in a HTML website to be in a container. This container has spefic properties that define the dimensions of the container. It helps with the design of a website and the allocation of objects. There is essentially a contatiner for every HTML element. It consists of: content, padding, borders and margins. The image below illustrates the container diagram:
Content - This is the inn part of the container and can be image or text. heading, etc.
Padding - The inner space between the content and the border of the container.
Border - The border of the container.
Margin - The outer space surrounding the container.
